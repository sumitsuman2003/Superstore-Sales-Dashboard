# Superstore-Sales-Dashboard
Excel Data Analyst Project - Superstore Sales Dashboard


## Project Overview
This project analyzes Superstore sales data using Excel.

## Tools Used
- Excel
- Pivot Table
- Charts
- Dashboard

## Dashboard Includes
- Monthly Sales Trend
- Sales by Category
- Sales by Region
- Top 10 Products

## Insights
- Technology category has highest sales
- West region performs best
- Sales increasing year by year
